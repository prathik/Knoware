About

How to use Knoware?

  • Add an idea by going to Add Idea! page.
  • Set the date on which it was posted on and review time.
  • On the end of the review time you will be sent an email asking for reviewing the idea.
  • You can the assign a status as approved or rejected for an idea.
  • Approved ideas can be used and tracked.
  • Rejected ideas can be tracked and ensured that it is not used.
  • Rejected ideas can be sent back to review state.

Development

Knoware is built using AngularJs and the backend is build using PHP. Knoware uses Bootstrap for front end design.

Knoware is open source, you can contribute at https://github.com/prathik/knoware.
